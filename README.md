<h1 align="center">TypeScript</h1>
<p>
This is a basic TypeScript project with Node.js, for the database we will use MySql and we will create a simple CRUD for the users table.
</p>

### Topics covered

* Creating projects with TypeScript
* Configuring Express with TypeScript
* Connecting to MySQL from Node
* Creating a CRUD using Sequelize
